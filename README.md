# WHMCS Models
Adiciona funcionalidades aos Models do WHMCS

## Instalação

Via Composer

*Biblioteca em desenvolvimento*

```sheel
composer require cesarlopes/whmcs-models:dev-main
```

```sheel
composer require cesarlopes/whmcs-models
```

## Utilização

*Biblioteca em desenvolvimento*

```php
require __DIR__.'/vendor/autoload.php';
```
